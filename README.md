# 東京湾ビシアジ 船別釣果アプリ

日付を指定すると、次を1画面で表示する Flask アプリです。

- 東京湾ビシアジの船別釣果
- 観音崎の潮位イベント
- 観音崎の潮位変化グラフ
- 観音崎の天気・最高気温・最低気温

## 対象船宿

今回の版では、次の船宿を対象にしています。

- 教至丸
- 海福丸
- 関義丸
- おかだ丸
- 五郎丸
- かもい丸
- 巳之助丸

## 画面の使い方

1. アプリを開く
2. 画面上部の `日付` 欄をタップする
3. 見たい日付を選ぶ
4. `表示する` を押す
5. その日の結果が下に表示される

表示される内容は次の通りです。

- 船別釣果
  - 船種
  - 魚種
  - サイズ
  - 釣果
  - 場所・水深
  - コメント
- 観音崎の潮汐イベント
  - 満潮・干潮の時刻
  - 潮位
  - 潮回り
- 観音崎の天気
  - 天気
  - 最高気温
  - 最低気温
  - 正午気温

## iPhoneで使ういちばん簡単な方法

1. Render に公開する
2. 公開URLを iPhone の Safari で開く
3. Safari の共有ボタンを押す
4. `ホーム画面に追加` を選ぶ
5. ホーム画面から開く

これでアプリのように使えます。

## ローカルで動かす方法

### Mac / Linux

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

### Windows

```bash
python -m venv .venv
.venv\Scriptsctivate
pip install -r requirements.txt
python app.py
```

起動後、ブラウザで `http://127.0.0.1:5000/` を開きます。

## Render に公開する方法

1. このフォルダを GitHub にアップロードする
2. Render で `New Web Service` を選ぶ
3. GitHub のこのリポジトリを選ぶ
4. `render.yaml` を使ってデプロイする
5. 公開URLが出たら Safari で開く
6. `ホーム画面に追加` で iPhone に置く

## 参照先

- 船宿釣果: 教至丸、海福丸、関義丸、おかだ丸、五郎丸、かもい丸、巳之助丸
- 潮位: Fishing Japan / chowari の観音崎潮見表ページ
- 天気: Open-Meteo API

## 注意

- 船宿サイトのHTML構造が変わると、抽出結果が崩れることがあります。
- 船宿によっては最新数日分のみ公開で、指定日のデータが見つからないことがあります。
- 潮位グラフは満潮・干潮の公表値から補間した参考グラフです。
- さらに船宿を増やす場合は `BOAT_SOURCES` に追加してください。
